# GrayChanCountDown
GrayChanCountDown is plugin for UnrealEngine4. 
---

# Grayちゃんモデルの公開日までカウントダウン  

だけを表示する意味のない煽りかつプラグイン勉強用遊びプラグインです。  
期限日は2016/1/1として設定しています。  

深い意味はありません。  
本プラグインはUE4のPluginBrowserのTemplate機能を使ってベースのソースを作成しています。  


# 注意

**本プラグインは4.10用にビルドしていますが、パッケージビルドで本プラグインが有効な状態で実行すると100%失敗します。**  
このため、必ずパッケージビルド時には本プラグインを除外したあとで実施する用にお願いします。  

一応、UE4本家ソースのGithubを見る限り4.11ブランチ以降ではそれっぽい修正コミットが入っていそうですが、  
UE4.10環境までは含まれていないためどうあがいても無理でした。  

# 導入・操作手順

「Plugins」フォルダ毎導入したいプロジェクトにコピーすればOKです。  
プラグインを有効にした状態でエディタを再起動。  
起動後にエディタのツールバーに「GrayChanCD」のボタンをクリックすると新しいタブが表示されます。  
それだけです。
